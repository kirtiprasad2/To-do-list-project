
# To-Do List App 📝

A simple and beautiful To-Do List web app built using HTML, CSS, and JavaScript. This project allows users to add, delete, and manage tasks with an attractive interface

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive design
- Clean background UI

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 📁 Project Structure

index.html
styles.css
script.js
